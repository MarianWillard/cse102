One Bit Memory with NOR

This circuit uses NOR gates to store the value of one bit. The value is represented by the LED being on as a 1 and off as a 0.
NOR gates are equivalent to NOT OR. NOR outputs the value of an OR gate followed by a NOT gate. Thus, NOR gates output 1 only if both inputs are 0.
PushOn devices are 1 only when they are being held on, so clicking down. They are 0 otherwise.
Notice how the LED turns on when the top PushOn is clicked and stays on when the PushOn is released. The bottom PushOn turns off the LED.

One Bit Memory with NOR

This circuit stores the value of one bit using only NAND gates.
NAND gates are equivalent to NOT AND, AND followed by a NOT. Thus, NAND output 1 if either or both of the inputs is 0. Following the circuit, one can see how the value is stored.
